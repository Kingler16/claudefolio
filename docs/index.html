Setup

Up and Running in 5 Minutes

No Docker, no cloud, no complexity.

1

Clone & Setup

The interactive wizard handles everything — language, tax regime, Telegram bot, API keys, portfolio import.

git clone https://github.com/Kingler16/Velora.git && cd Velora && python3 setup.py
2

Run Your First Briefing

Collects market data, runs AI analysis, sends the result to your Telegram.

python -m src.main briefing
3

Start the Dashboard

Open the web dashboard and explore your portfolio visually.

python -m src.main web
4

Deploy (Optional)

Put it on a Raspberry Pi for always-on briefings and dashboard access from any device.

./scripts/deploy.sh admin@your-pi-ip
